# branding/

Visual identity assets, added in **Phase 1 (MVP respin)**.

- `logo/` — Pacific Linux logo, source files + exported PNG/SVG
- `wallpapers/` — default desktop wallpaper(s), 1920x1080 minimum
- `plymouth/` — boot splash theme (Plymouth)
- `icons/` — icon theme overrides / custom app icons

Keep source files (`.svg`, `.xcf`, `.afdesign`, whatever the tool) alongside
exports so assets stay editable.
